## 📋 RESUMEN DE SOBERANÍA (MARZO 2026)

2.  **DBCI-SSOT (lfc-terminology.js)**: Cerebro central del proyecto. Si un dato cambia en el SSOT, se propaga por toda la ingeniería.
3.  **La Cocina vs El Restaurante**:
    *   **I-VI (La Cocina)**: Ingeniería de fuente (Markdown) y Decisiones Técnicas (DT) constantes.
    *   **Safe-Cooker (lfc-cli.js)**: Motor de generación que garantiza la pureza y el estilo L4.
    *   **X (El Restaurante)**: Entregables consolidados para el Cliente/Stakeholders.

## ⚙️ **EL MOTOR LFC-CLI (MASTERCHEF v6.5)**

El núcleo de este repositorio es `scripts/lfc-cli.js`. Sus funciones principales son:

*   **`purify`**: Auditoría forense automatizada que elimina términos obsoletos (ej. 25kV, GSM-R, Eurobaliza) y los reemplaza por la terminología soberana.
*   **`serve`**: Proceso de "cocinado" que genera HTML premium y Word oficial sincronizando el SSOT.
*   **`karpathy-loop`**: Ciclo de mejora continua que detecta inconsistencias mediante auditoría recursiva.

## 🔍 **REFERENCIAS CRUZADAS Y TRAZABILIDAD**

El sistema utiliza **IDs de Referencia** para asegurar que cada cantidad (15 locomotoras, 37 estaciones, 526 km) tenga una justificación técnica atómica en los documentos de ingeniería conceptual y de detalle.

3.  **Auditoría**: El `karpathy_audit.js` valida que no queden "zombis" de versiones anteriores.
