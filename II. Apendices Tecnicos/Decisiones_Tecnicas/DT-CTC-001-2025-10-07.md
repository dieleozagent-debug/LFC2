## 3. JUSTIFICACIÓN TÉCNICA

### Lógica de dependencia:
```
1 Computadora PTC embarcada (1.2.100)
    ↓
Requiere:
  - 1 Licencia de Software PTC (1.2.104)
  - 1 Interface de frenos (1.2.105)
  - 1 Sistema de redundancia (1.2.106)
```

**Si hay 8 computadoras PTC, entonces:**
- Software PTC: 8 licencias (no 15)
- Interfaces frenos: 8 módulos (no 15)
- Sistemas redundancia: 8 sistemas (no 15)

### Justificación técnica por item:

#### 1.2.104 - Software PTC:
- Cada computadora PTC requiere **1 licencia** de software
- El software controla: velocidad, frenado, protección, comunicaciones
- **Cantidades:** 8 computadoras = 8 licencias

#### 1.2.105 - Interfaces frenos:
- Cada computadora PTC requiere **1 interface** con el sistema de frenos
- La interface traduce comandos digitales (PTC) en actuación física (frenos)
- **Cantidades:** 8 computadoras = 8 interfaces

#### 1.2.106 - Sistema redundancia:
- Cada sistema PTC requiere **1 módulo** de redundancia (procesador de respaldo)
- Cumple requisitos SIL-4 (duplicación de procesamiento crítico)
- **Cantidades:** 8 computadoras = 8 sistemas redundancia

### Alineación con DT-TETRA-001:
Esta DT **complementa** la DT-TETRA-001, ajustando los ítems que quedaron pendientes para lograr **coherencia total en la sección 1.2 (Sistema PTC Embarcado)**.
