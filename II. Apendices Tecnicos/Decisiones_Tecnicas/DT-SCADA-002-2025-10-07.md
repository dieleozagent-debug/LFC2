## 2. OBSERVACIONES TÉCNICAS

### **Razón del cambio:**
Actualización del precio del software SCADA para reflejar las capacidades avanzadas requeridas para gestión integral del corredor ferroviario de 526 km con arquitectura distribuida y redundante.

### **Alcance técnico del software SCADA:**

**Módulos principales incluidos:**
1. **Supervisión en tiempo real**
   - Monitoreo 37 estaciones TETRA/Red Vital IP / TETRA
   - Seguimiento 8 trenes con PTC embarcado
   - Gestión 5 enclavamientos ENCE
   - Control 146 pasos a nivel (9 Tipo C, 15 Tipo B, 122 otros)

2. **Control distribuido**
   - 25 desvíos motorizados remotos
   - 73 cámaras CCTV PTZ 4MP
   - Sistemas de seguridad y acceso

3. **Gestión de datos históricos**
   - Almacenamiento mínimo 10 anos
   - Trending y análisis predictivo
   - Reportes regulatorios automáticos

4. **Redundancia y alta disponibilidad**
   - Arquitectura N+1
   - Failover automático < 2 segundos
   - SLA 99.99%

5. **Integración y protocolos**
   - OPC-UA, DNP3, IEC 60870-5-104
   - Modbus TCP/IP, SNMP
   - APIs RESTful para integraciones externas

6. **Ciberseguridad**
   - IEC 62443-3-3 compliant
   - Autenticación multifactor
   - Encriptación end-to-end
   - SIEM integrado

7. **Módulos especializados ferroviarios**
   - Gestión de tráfico ferroviario (TMS)
   - Planificación de horarios
   - Gestión de incidentes
   - Mantenimiento predictivo
